Analyze HTTP basic authentication

HTTP is an application layer protocol that is designed for communication between web browsers and web servers. HTTP is unencrypted. It doesn’t even encrypt some parts as RADIUS does in the passwords. So it is really not secure. HTTP runs on port 80, unlike HTTPS that runs on port 443 and is a secure protocol as it uses TLS.

An HTTP basic authentication is a simple challenge and response mechanism with which a server can request authentication information(a user ID and a password) from a client. A public web server also needs to be used that hosts an HTTP page. The username the web server will be waiting for is “username” and the password is “password”

The public web server used here is hosted at this url: httpbin.org/basic-auth/username/password

The packet capture needs to be initiated in Wireshark first. The capture filter to use here is port 80 and by double-clicking the network interface initiated the capture.

For the first case, the username is put as “username” and the password as “1234”. As the password is not the intended password, by clicking on Sign in, it will prompt again for the credentials. Now when tried with the correct credentials “username” and “password”:

As we can see here that this time it is the correct response.

Going back to Wireshark, we stop capturing the traffic.

Again it is seen that the communication is unencrypted. The username and password can be easily seen by analyzing the packets. To do this we click on the first HTTP GET packet > expand Hypertext Transfer Protocol > expand Authorization

Here beside the word Basic there is the base 64 encoding of the password, which does not provide any encryption of the username and the password being sent to the server. Wireshark decoded the base 64 encoding, so now easily the username and the incorrect password that is being sent can be seen right there on the middle pane.

Similarly, the second authentication attempt can also be analyzed now, but this time the username is seen along with the correct password.
